Education Perfect Script Break Down

Hi all, thanks for having a look at this project of mine. The purpose of this script is to help students do some of that tedious work no-one wants to do on Education Perfect.
Below are the main features of this application:
1) Anti-Snitch this feature once turned on will disable the EP feature which can detect when you are off the screen. That means you can play a game in a different tab and the teacher will be none the wiser.
2) Works even in exam mode (self explanitory)
3) Quick hide feature if you see the teacher coming you can simply click the x to make it disapear, when you want it again click the little thunderbolt.
